Zer: Zoom Enhance Rotate

A framework for manipulating 2D scenes and providing a simple contract to build 2D editors. It's primary goal is to support World Bootstrap by simplifying and wrapping JavaFX. Please head on over to World Bootstrap for more information.

See https://github.com/jeffrey-io/zer for more information.
